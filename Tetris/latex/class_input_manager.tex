\doxysubsection{Detailed Description}
Gestionarea input-\/urilor utilizatorului pentru jocul Tetris. 

\doxylink{class_input_manager}{Input\+Manager} permite înregistrarea și verificarea input-\/urilor pentru diferite tipuri definite în Tetris\+API\+::\+EInput\+Type. Stochează tastele și stările acestora și le verifică în timpul execuției.
