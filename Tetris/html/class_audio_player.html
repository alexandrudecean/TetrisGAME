Detailed Description

Clasa pentru gestionarea efectelor sonore și muzicii din joc.

AudioPlayer moștenește funcționalitățile clasei BaseObserver și implementează răspunsurile la evenimentele jocului. Gestionarea sunetelor include muzica de fundal, sunetul la rotație, sunetul pentru ștergerea liniilor și sunetul pentru sfârșitul jocului.
